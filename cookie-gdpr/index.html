Kebijakan Cookie GDPR

Selamat datang di blitiket.id (selanjutnya disebut "Kami," "Kami," atau "Website"). Kami menggunakan teknologi cookie untuk meningkatkan pengalaman Anda saat menggunakan situs web kami.

Dengan menggunakan situs web kami, Anda setuju untuk menggunakan cookie sesuai dengan Kebijakan Cookie ini. Jika Anda tidak setuju dengan penggunaan cookie ini, Anda dapat menonaktifkannya dalam pengaturan browser Anda atau tidak menggunakan situs web kami.

Apa itu Cookie?

Cookie adalah file kecil yang disimpan di perangkat Anda yang mengandung informasi tentang kunjungan Anda ke situs web. Cookie digunakan untuk berbagai tujuan, termasuk untuk mengingat preferensi Anda, melacak aktivitas Anda, dan menganalisis bagaimana Anda berinteraksi dengan situs web.

Jenis Cookie yang Kami Gunakan

Kami menggunakan cookie berikut di situs web kami:

  • Cookie Analitik: Cookie ini digunakan untuk menganalisis bagaimana pengguna berinteraksi dengan situs web kami dan membantu kami meningkatkan kinerja situs web.
  • Cookie Fungsional: Cookie ini digunakan untuk mengingat preferensi Anda, seperti bahasa yang Anda pilih dan pengaturan lainnya.
  • Cookie Pihak Ketiga: Kami juga bekerja sama dengan pihak ketiga yang dapat mengatur cookie di perangkat Anda. Kami tidak mengontrol cookie pihak ketiga ini dan Anda harus merujuk ke kebijakan privasi masing-masing pihak ketiga untuk informasi lebih lanjut.

Bagaimana Mengelola Cookie

Anda dapat mengelola pengaturan cookie Anda melalui pengaturan browser Anda. Kebanyakan browser memungkinkan Anda untuk menolak atau menghapus cookie. Namun, mematikan cookie atau mengatur pengaturan cookie tertentu dapat memengaruhi fungsionalitas situs web kami.
